Automatically updates the TYPO3 CMS core to its latest minor release if these criteria are met:

  • It only works under Unix and MacOS (support of symbolic links is required).
  • typo3_src must be a symbolic link to the extracted TYPO3 source directory.
  • The symbolic link needs to be writable (and deletable) by the web server user.
  • The document root directory needs to be writable.
  • One path above document root (../) needs to be writable (new directories will be created).
  • The tar command must be available for extracting the TYPO3 source package.

Composer Mode is active. Please update using composer commands.
 The environment variable was set TYPO3_DISABLE_CORE_UPDATER=1.
 This system uses an unreleased TYPO3 core version.
 This action can only be used with a linked typo3_src.
