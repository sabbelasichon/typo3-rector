===============================================================
Feature: #92366 - Show fragments in preview of InputLinkElement
===============================================================

See :issue:`92366`

Description
===========

The rendertype `InputLinkElement` renders a preview of the link since :issue:`28171`.

If a link to a page contains a fragment, this information has been added to the preview.


Impact
======

Editors are able to see the anchor information of a link without toggling the wizard.
