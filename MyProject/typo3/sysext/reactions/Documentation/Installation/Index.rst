============
Installation
============

For :ref:`Composer-based TYPO3 installations <t3start:install>`:

..  code-block:: bash

    composer require typo3/cms-reactions

For :ref:`legacy installations <t3start:legacyinstallation>` activate the
:guilabel:`TYPO3 CMS Reactions` system extension in
the :ref:`Extension Manager <t3start:extensions_legacy_management>`.
